# Guide d'auteur

Une note Markdown est une unité éditoriale autonome. Elle peut viser un thème, une espèce, une plantation,
une zone, une section, un scan, ou plusieurs à la fois.

Le corps contient le texte rédigé.
Le front matter contient les métadonnées de compilation.

Les profondeurs de texte attendues sont :
- Carte
- Court
- Moyen
- Long
- Notes internes

Les rôles de rattachement possibles dans `attachments` :
- primary
- contextual
- example
- located_in
- scan_entry
- belongs_to

Les relations entre nœuds vont dans `edges`.
