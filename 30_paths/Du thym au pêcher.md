## Carte

Un jardin peut se lire comme une promenade intellectuelle.

## Court

Ce parcours propose trois étapes : le thym, les strates, le pêcher.

## Moyen

Le thym permet d'entrer dans le jardin par la pluralité des fonctions. Les strates permettent
de comprendre comment cette pluralité s'organise dans l'espace. Le pêcher réintroduit enfin
la promesse gourmande.

## Long

Le parcours n'est pas une simple suite de contenus. C'est une manière de régler le regard.
Commencer par le thym, c'est refuser d'entrer seulement par ce qui se voit le plus.
